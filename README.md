# MediLink - Connecting Care
### IIT Patna Capstone Project

MediLink is a secure healthcare platform that connects patients and doctors.

**Key Features:**
* **Authorization Handshake:** Secure data sharing protocol.
* **Vitals Analytics:** Real-time trend plotting using Matplotlib.
* **Emergency QR:** Scan-to-access Patient ID.
* **File Vault:** Secure upload and management of medical reports.

**Tech Stack:** Python, Flask, SQLAlchemy, Matplotlib, QRcode.
